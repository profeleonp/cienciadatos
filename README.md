# cienciadatos

Para un conjunto de datos fijos provenientes de un estudio, he creado en éste repositorio una hoja de cálculo en Excel con un ejemplo de 60 datos donde el diagrama de caja y bigote permite detectar datos atípicos o anómalos.

Para datos de una serie de tiempo, recomiendo seguir el video tutorial del profesor David Martín donde explica la detección de anomalías en una secuencia de dato a través del tiempo.
[Detección de Anomalías con Python Pandas](https://www.youtube.com/watch?v=6TyepQq9NBc).
